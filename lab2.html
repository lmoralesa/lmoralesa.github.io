Lab #2: Information Gathering

Introduction

Information gathering is a crucial initial phase of penetration testing. The goal is to collect as much information as possible about the target system or network. This information can include usernames, IP addresses, server names, and any other relevant data.

Active vs. Passive Information Gathering

Information gathering techniques are broadly categorized as either active or passive.

  • Active Information Gathering: Sending traffic to the target network (e.g., ping sweeps, port scans).
  • Passive Information Gathering: Collecting information without directly interacting with the target (e.g., using search engines, looking at publicly available information).

Publicly Available Information

Many sources of information are publicly available on the internet. Using these sources minimizes the risk of detection.

  • The Wayback Machine: https://web.archive.org/
  • Netcraft: https://searchdns.netcraft.com/
  • CentralOps: https://centralops.net/co/
  • ViewDNS: https://viewdns.info/

Collecting Information from Documents

Tools like Metagoofil can extract metadata from publicly available documents (PDF, DOC, XLS, etc.).
