Lab #1: Security Principles & Kali Linux Installation

Introduction to Penetration Testing

Penetration testing is a process used to conduct high-level security assessments and audits. A methodology defines the rules, practices, procedures, and methods to follow during a security audit.

Types of Penetration Tests

Common penetration testing types include black-box, white-box, and gray-box testing.

  • Black-box testing: The tester has no prior knowledge of the system.
  • White-box testing: The tester has complete knowledge of the system.
  • Gray-box testing: The tester has partial knowledge of the system.
