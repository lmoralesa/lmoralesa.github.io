Gathers emails, subdomains, hosts, employee names from public sources — Google, LinkedIn, Shodan, GitHub.
theHarvester -d target.com -b all

Gobuster / Dirb
Brute-force URI directories, DNS subdomains, and virtual host names against web servers at high speed.
gobuster dir -u http://target -w /usr/share/wordlists/dirb/common.txt

Fast web fuzzer written in Go — directory, parameter, header, and vhost fuzzing with advanced filtering.
ffuf -w wordlist.txt -u http://target/FUZZ -mc 200

Home Lab

My personal cybersecurity lab runs 24/7 as a hands-on environment for testing, monitoring, and practicing defensive and offensive techniques. Every tool is self-hosted and actively maintained.

🔓
Pi-hole
DNS Sinkhole Ad Blocking
Network-wide DNS-based ad and tracker blocker. Intercepts all DNS queries on the LAN, blocks malicious domains, and provides full DNS query logging for traffic analysis and threat detection.
DNS DHCP blocklist Raspberry Pi query logs
🛡
AdGuard Home
DNS over HTTPS DNS-over-TLS
Advanced DNS filtering server with support for DoH and DoT encrypted protocols. Used as a secondary DNS resolver with custom filtering rules, parental controls, and encrypted upstream resolvers.
DoH DoT custom rules encrypted DNS rewrites
🔥
pfSense
Firewall VPN Gateway BSD
Open-source firewall and router platform. Manages network segmentation, VLAN isolation, stateful packet inspection, IDS/IPS via Suricata, OpenVPN gateway, and traffic shaping for the entire home lab.
VLAN NAT Suricata OpenVPN pfBlockerNG WireGuard
🔐
OPNsense
Next-Gen FW IDS/IPS HardenedBSD
HardenedBSD-based firewall with a modern web UI. Used in a secondary lab segment for testing firewall rules, deep packet inspection, HAProxy load balancing, and comparing security postures vs pfSense.
Zenarmor HAProxy Unbound DNS 2FA GeoIP blocking

Fail2ban
Brute-Force Protection IP Banning
Intrusion prevention framework that monitors log files and bans IPs after repeated failed login attempts. Configured to protect SSH, web services, and exposed APIs against brute-force and credential stuffing attacks.
SSH protection iptables nginx jails auto-ban log parsing

Portainer
Container Mgmt Docker / K8s
Web-based GUI for managing Docker containers, images, volumes, and networks across all lab nodes. Used to deploy and monitor self-hosted services, test container isolation, and practice Kubernetes workload management.
Docker Kubernetes Stacks RBAC image scanning Swarm

Zabbix
SIEM-lite Monitoring Alerting
Enterprise-grade open-source monitoring platform. Monitors all lab hosts, containers, network devices, and services in real time. Configured with custom triggers for security anomalies, port scans, and unauthorized access attempts.
SNMP agents dashboards triggers SMTP alerts API

Grafana + Prometheus
Dashboards Metrics
Prometheus scrapes metrics from all lab services and containers; Grafana visualizes network throughput, CPU load, blocked DNS queries, firewall drops, and security events in real-time custom dashboards.
PromQL node exporter alerting Loki log aggregation

Open-source network graphing and monitoring solution built on RRDtool. Used to poll SNMP data from routers, switches, firewalls and servers — generating historical performance graphs for bandwidth, CPU, memory, and interface utilization across all lab segments.
