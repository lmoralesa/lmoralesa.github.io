What is Wireshark?

Wireshark is a free and open-source packet analyzer. It's used for network troubleshooting, analysis, software and communications protocol development, and education.

Common Uses

  • Network troubleshooting and analysis
  • Security analysis and packet inspection
  • Protocol development and debugging
  • Network monitoring and diagnostics

Installation Guide

  1. 1. Visit the official Wireshark website: Download Page
  2. 2. Select your operating system (Windows, macOS, or Linux)
  3. 3. Download and run the installer
  4. 4. Follow the installation wizard steps
  5. 5. Install WinPcap/Npcap if prompted (required for packet capture)

Kali Linux Installation Guide

What is Kali Linux?

Kali Linux is an advanced penetration testing and security auditing Linux distribution. It comes pre-installed with numerous security and forensics tools, making it the preferred choice for cybersecurity professionals and ethical hackers.

Key Features:
  • 600+ Pre-installed Tools
  • Open Source
  • Multi-language Support
  • Regular Security Updates

Prerequisites

  • A computer with at least 4GB RAM (8GB recommended)
  • 20GB of free disk space (40GB recommended)
  • VirtualBox installed on your system
  • Kali Linux VirtualBox image downloaded

Installation Steps

  1. 1. Install VirtualBox
    • Download VirtualBox for your operating system
    • Run the installer and follow the installation wizard
    • Accept the default settings during installation
  2. 2. Download Kali Linux
    • Visit the Kali Linux downloads page
    • Select "Virtual Machines"
    • Download the VirtualBox image
  3. 3. Import Kali Linux to VirtualBox
    • Open VirtualBox
    • Click "Import"
    • Select the downloaded Kali Linux .ova file
    • Adjust RAM (2GB minimum) and CPU settings if needed
    • Click "Import" and wait for the process to complete
  4. 4. Start Kali Linux
    • Select the imported Kali Linux VM
    • Click "Start"
    • Default credentials: username: kali, password: kali
  5. 5. Post-Installation
    • Update the system: sudo apt update && sudo apt upgrade
    • Install VirtualBox Guest Additions for better performance
    • Configure screen resolution and shared folders if needed
